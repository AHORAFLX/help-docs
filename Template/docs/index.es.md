# Empezar a desarrollar

Esta plantilla de MkDocs proporciona un marco para crear documentación profesional y multiidioma. Esta guía cubre la configuración, creación de contenido y despliegue para productos, APIs y procesos internos.

### Primeros Pasos
Todo lo que necesitas para comenzar a crear documentación:

- **[Inicio rápido](GettingStarted/QuickStart.md)** - Guía básica de configuración
- **[Configuración de navegación](GettingStarted/VisualStructure.md)** - Cómo se generan los menús
- **[Organización general](GettingStarted/FileStructure.md)** - Estructura de carpetas, metodología de trabajo básica

### Escribir Contenido

**Nuestros componentes:**

- **[Componentes personalizados](Components/CustomComponents.md)** - Elementos interactivos
- **[Clases personalizadas](Components/CustomClasses.md)** - Opciones especiales de estilo

### Despliegue

- **[Construir tu sitio](Deployment/BuildingYourSite.md)** - Compilar para producción

### Solución de problemas
